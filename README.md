# VBA_Tool_ScoreResult
SubプロシージャとFunctionプロシージャを組み合わせて、架空の試験結果データから

3教科の平均点算出と合格不合格の判定を自動化するVBAツール。


（ScoreResult 概要）

・3教科の平均点算出

・3教科の平均点から合格不合格の判定

上記2つの処理をFunctionプロシージャに分けて作成。

Subプロシージャは加工対象データの読み込み ～「成績一覧」シート出力を実行。


(ScoreResult 実行手順)

1, このリポジトリに添付されている拡張子「.xlsm」で終わるファイル内の「試験結果」シートをコピーして新たに新規ブックを立ち上げて

新規ブック内のシートへコピーした「試験結果」シートを貼り付ける。シート名は「試験結果」に変更する。

2, このリポジトリに添付されている「.bas」ファイルの内容をコピーし、1で作成したブック内で「Alt + F11」を入力して表示された画面上部の

「挿入」から「標準モジュール」を選択する。

3, 選択して表示された画面に、2でコピーした「.bas」ファイルの内容を貼り付ける。

4, 画面上部の三角の実行ボタンを選択。


「VBAコード詳細は別添付ファイル(.bas)を参照。」
